What is Git?

Git is a free and open-source distributed version control system that helps developers track changes in source code.

  • ✅ Keeps track of code history
  • ✅ Allows collaboration among developers
  • ✅ Works offline and locally

What is GitHub?

GitHub is a cloud-based hosting service that lets you manage Git repositories online.

  • ✅ Share code with others
  • ✅ Work with teams worldwide
  • ✅ Provides project management tools
Visit GitHub

Common Git Commands

Here are some essential Git commands:

git init         → Initialize repository
git clone URL    → Clone a repo
git status       → Check status
git add .        → Stage changes
git commit -m "" → Save changes
git push         → Upload to GitHub
git pull         → Download updates
